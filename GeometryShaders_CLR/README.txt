# Geometry-Shaders

Controls:

1: Spawn a clean explosion.
2. Spawn a dirty explosion.
3. SPawn rain.

Numpad1: Change Shader to billboard.
Numpad2: Change shader to Cubes.
Numpad3: Change to two cubes. Smaller one orbiting the larger.

P: Pause
M: Show/Hide the model
T: Detonate/Reset the model. 
R: Reverse the explosion of the model.
X: Stop rendering to screen.
F: Show/Hide FPS.
C: Lock/Unlock mouse from window.
ESC: Exit application.
